## Serializace (a deserializace) binárního (vyhledávacího) stromu

Přednáška odpadla, pouze opakováni toho, co je binární vyhledávací strom a jeden lehce naivní způsob, jak ho implementovat.
Součástí implementace je i funkce pro serializaci. Je fajn si rozmyslet, že serializační funkce nijak nevyužívá "vyhledávací" vlastnosti stromu, otevírají se tedy lepší možnosti serializace!

Je taky fajn si zkusit napsat, nebo alespoň rozmyslet, jak strom realizovat bez rekurze.
